# LLM Research Paper Summarizer

## Overview
This project allows users to upload research papers in PDF format, generate summaries, and ask questions about the paper using Retrieval-Augmented Generation (RAG).

## Features
- PDF Upload
- Automatic Summary Generation
- Semantic Search using ChromaDB
- Question Answering
- Streamlit Web Interface

## Technologies Used
- Python
- Streamlit
- Sentence Transformers
- ChromaDB
- Hugging Face Transformers
- PyMuPDF

## How to Run

pip install -r requirements.txt

streamlit run app.py
